Hashtaurante

Aplicação web de cardápio digital interativo, desenvolvida com React e Vite,
inspirada em interfaces modernas de restaurantes.
O projeto tem foco em organização de layout, componentização e boas práticas de desenvolvimento front-end, proporcionando uma experiência clara e responsiva ao usuário.

Tecnologias utilizadas:
React
Vite
JavaScript (ES Modules)
HTML5
CSS3
ESLint

Funcionalidades:
Exibição de categorias de produtos
Listagem de itens em formato de cards
Layout responsivo para diferentes tamanhos de tela
Estilização utilizando CSS Grid e Flexbox
Estrutura de projeto organizada e escalável

Como executar o projeto:
Clonar o repositório
git clone https://github.com/seu-usuario/hashtaurante.git

Acessar o diretório do projeto:
cd hashtaurante

Instalar as dependências:
npm install

Executar em ambiente de desenvolvimento:
npm run dev

A aplicação estará disponível em http://localhost:5173.
